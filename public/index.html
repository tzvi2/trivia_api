Welcome to my trivia game API

There are only two parameters for each request: topic, and number of questions For example, if you wanted 5 questions about animals, you'd make a get request to:

https://api4trivia.herokuapp.com/questions/animals/5

Available Topics

  • Biology (.../questions/biology)
  • U.S. History (.../questions/us_history)
  • Sports (.../questions/sports)
  • Geography (.../questions/geography)

If you wanted 10 questions about sports, the full URL to call is:

https://api4trivia.herokuapp.com/questions/sports/10

Response format

You'll always sent back an array of objects. Each object will have a question key, as well as an answers key. For example: fetch(https://api4trivia.herokuapp.com/questions/sports/2) will get the response: 

    [
        {
          "question": "In what year was the NFL founded?",
          "answers": [
            {
              "text": "1820",
              "correct": false
            },
            {
              "text": "1920",
              "correct": true
            },
            {
              "text": "1980",
              "correct": false
            },
            {
              "text": "1992",
              "correct": false
            }
          ]
        },
        {
          "question": "The Tampa Bay Rays were originally called...",
          "answers": [
            {
              "text": "Sun Rays",
              "correct": false
            },
            {
              "text": "Pay Rays",
              "correct": false
            },
            {
              "text": "Devil Rays",
              "correct": true
            },
            {
              "text": "Hot Rays",
              "correct": false
            }
          ]
        }
    ]
Each question contains four answers total, three of which will be incorrect.
